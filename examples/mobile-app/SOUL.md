# SOUL.md

## the vibe

native-feeling, responsive, accessible. it should feel like a real app, not a website.

## design principles

- tap targets at least 44x44
- every interaction has haptic or visual feedback
- loading states are skeletons, not spinners
- offline support for core features
- first launch experience should be under 30 seconds

## anti-patterns

- no web-style hover states
- no horizontal scrolling on mobile
- no small text (minimum 16sp for body)
- no blocking the main thread with heavy work
