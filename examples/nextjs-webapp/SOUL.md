# SOUL.md

## the vibe

clean, minimal, fast. this app should feel like it gets out of the way.
no animations for the sake of animations. every interaction should feel instant.

## design principles

- ship fast, fix later
- clear over clever
- every page should load in under 2 seconds
- errors should tell the user what to do next
- mobile-first always

## tone

friendly but not chatty. confident but not corporate.
lowercase UI copy is fine. no exclamation marks unless something actually exciting happened.

## anti-patterns

- no "welcome to our platform" hero text
- no loading spinners over 2 seconds — use skeletons instead
- no modals when a page would do
- no dark patterns in auth flows
