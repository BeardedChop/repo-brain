# changelog

all notable changes to this project will be documented in this file.

## [0.2.0] - 2026-04-16

### added

- CI/CD workflow examples (GitHub Actions) for all three example projects
- pre-commit hook examples (Husky + lint-staged for JS, pre-commit for Python)
- dependency files: `package.json` for nextjs-webapp and mobile-app, `requirements.txt` for python-api
- Dockerfile and docker-compose.yml for the python-api example
- "choosing your stack" guide in the nextjs-webapp AGENTS.md (helps pick between auth/database options)
- expanded testing guidance with test patterns and examples in all three AGENTS.md files
- logging and monitoring section in SECURITY.md
- explanation of `import AGENTS.md` syntax in the README

## [0.1.0] - 2026-04-15

### added

- initial release
- six core template files: AGENTS.md, CLAUDE.md, SOUL.md, USER.md, TOOLS.md, SECURITY.md
- three example templates: nextjs-webapp, python-api, mobile-app
- getting-started guide in docs/
